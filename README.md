# OFÉK — Family Financial Planning

[Open the live app](https://gluckapixels.github.io/ofek-financial-app/)

OFÉK is a privacy-first Hebrew web application that helps families evaluate spending decisions and understand how today's choices may affect their path to financial independence.

## What it does

- Evaluates one-time, monthly and annual expenses.
- Models a family's target for financial independence.
- Separates liquid assets from retirement savings.
- Accounts for household size, income stability, housing, emergency savings, debt and expected retirement income.
- Shows the projected funding gap and a possible monthly adjustment.
- Stores the family profile locally in the browser — no account or server database is used.

## Technology

Vanilla JavaScript, semantic HTML, modern CSS, Vite and GitHub Pages.

## Important notice

OFÉK is an educational planning and illustration tool. It is not a substitute for personal financial, pension, tax or investment advice. Calculations are simplified projections and do not guarantee future results.
